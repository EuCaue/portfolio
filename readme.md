# Personal Portfolio

This is the repository for my personal portfolio website, built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. It is designed to showcase my skills, projects, and contact information in a clean, fast, and responsive way.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **E-mail Integration:** [EmailJS](https://www.emailjs.com/) / [@sendgrid/mail](https://sendgrid.com/)
